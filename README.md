# Titanic Data Cleaning & Preprocessing

## Objective
The objective of this project is to clean and preprocess the Titanic dataset for further analysis and machine learning applications.

## Dataset
Titanic Dataset (Kaggle)

## Tasks Performed

### Data Cleaning
- Handled missing values
- Removed duplicate records
- Renamed columns for better readability
- Saved cleaned dataset

### Exploratory Data Analysis
- Dataset overview
- Summary statistics
- Missing value analysis

### Visualizations
- Survival Distribution
- Gender vs Survival
- Age Distribution

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Female passengers had a higher survival rate.
- Most passengers were between 20 and 40 years old.
- Missing values were mainly present in Age, Cabin, and Embarked columns.

## Output
- Cleaned dataset (`titanic_cleaned.csv`)
- Visualizations
- Jupyter Notebook
